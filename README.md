# E-commerce Automation Toolkit

A full-stack portfolio project for automating e-commerce workflows.

## Overview
This app receives Shopify webhook events, stores them, and provides an admin dashboard for managing automation workflows and AI-powered content actions.

## Planned Features
- Shopify webhook receiver
- Event dashboard in Vue
- Event detail view
- AI-generated product descriptions and tags
- Rule-based automations
- Filtering and search
- Deployment-ready full-stack app

## Stack
- Vue 3 + Vite
- Node.js + Express
- PostgreSQL
- Prisma
- OpenAI API

## Goal
Build a realistic internal tool that demonstrates:
- API integration
- webhook handling
- database modeling
- admin UI development
- automation logic
- AI feature integration
